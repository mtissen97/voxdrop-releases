# Voxdrop — Public Releases

This repository hosts the public release artifacts for **Voxdrop**, a macOS voice-to-text app.

→ **Download:** [voxdrop.io](https://voxdrop.io)
→ **Latest release:** [VoicePilot.dmg](https://github.com/mtissen97/voxdrop-releases/releases/latest/download/Voxdrop.dmg)

## What is Voxdrop?

Dictate into any app on your Mac with a single hotkey. Apple Silicon native. 7-day Pro Trial on first launch — no credit card required.

The source code is private. Release notes for each version are documented in the [Releases](https://github.com/mtissen97/voxdrop-releases/releases) tab.

## System Requirements

- macOS 14.0 or later
- Apple Silicon (M1 or newer)
